 📸 What It Looks Like

Search any developer → instant beautiful dashboard:

- **Profile card** with avatar, bio, location and join year
- **4 stat pills** — repos, followers, total stars, total forks
- **Language donut chart** — colour-coded breakdown of every language used
- **Horizontal bar chart** — top languages ranked by usage
- **Commit activity chart** — 6-month green bar graph of activity
- **Top repo hero card** — spotlights their most starred repository
- **Repo grid** — clickable cards for top repositories
- **⚔️ Compare mode** — search two developers side by side with a head-to-head winner panel

🛠️ Built With

- **HTML5** — Structure
- **CSS3** — Styling, animations, gradients, responsive layout
- **Vanilla JavaScript** — All logic, API calls, DOM manipulation
- **Chart.js** — Language donut chart, bar chart, activity chart
- **GitHub REST API** — All data is fetched live in real time
- **Google Fonts** — Bebas Neue + Outfit typefaces

No React. No Vue. No Node. No build tools. Just clean, readable code in a single file.


🚀 Getting Started

 Option 1 — Just open it (easiest)
1. Download `github-dashboard.html`
2. Open it in any browser
3. Done ✅

 Option 2 — Run with VS Code Live Server
1. Install [VS Code](https://code.visualstudio.com/)
2. Install the **Live Server** extension by Ritwick Dey
3. Open `github-dashboard.html` in VS Code
4. Click **Go Live** in the bottom right corner
5. App opens in your browser with live reload

Option 3 — Host it yourself (GitHub Pages)
1. Fork this repository
2. Go to **Settings → Pages**
3. Set branch to `main` → Save
4. Your app is live at `https://YOUR-USERNAME.github.io/gitpulse-dashboard/`


📖 How to Use

Single Profile Mode
1. Type any GitHub username in the search bar
2. Press **Enter** or click **SEARCH GITHUB →**
3. Explore the stats, charts and repos

Compare Mode ⚔️
1. Click the **⚔️ Compare** button in the top right
2. Enter two GitHub usernames
3. Press search — see them side by side with a winner panel

💡 Fun usernames to try
- `torvalds` — Creator of Linux
- `gaearon` — Creator of React
- `microsoft` — Microsoft's org
- `google` — Google's org
- Your own username!


⚠️ API Rate Limits

GitPulse uses GitHub's **free public API** with no authentication required.

| Limit | Detail |
|---|---|
| 60 requests/hour | Per IP address, no token needed |
| Resets every hour | Automatically |
| How to fix if hit | Wait a few minutes and try again |

For heavier usage, you can add a [GitHub personal access token](https://github.com/settings/tokens) to the fetch headers to increase limits to 5,000 requests/hour.
